# 👥 ConnectLedger — Smart Friendship & Interaction Manager

## 🚀 Project Overview

ConnectLedger is a professional friendship management dashboard designed to help users maintain meaningful social connections in a busy world.

It enables users to track when they last contacted friends, set relationship goals, and log every interaction—whether it’s a quick text or a long video call.

This project demonstrates full-stack front-end capabilities including Next.js App Router navigation, centralized state management for activity logging, and data visualization for social habit tracking.

---

## 🛠️ Tech Stack

- Next.js (App Router) – Routing and SSR optimization
- Tailwind CSS – Utility-first styling
- DaisyUI – Prebuilt UI components
- Recharts – Data visualization (analytics dashboard)
- React Context API – Global state management (interaction ledger)
- Lucide / React Icons – Icon system for UX clarity

---

## ✨ Key Features

### 🧑 Friend Management

Browse a curated list of friends with realistic profiles, enriched with bio, tags, and contact information. Each friend includes visual status indicators (Overdue, Almost Due, On Track) that dynamically reflect communication urgency.

### ⚡ Real-Time Interaction Logging

Users can log Calls, Texts, and Video interactions instantly using quick-action buttons. Every action triggers toast notifications and updates the global state in real time.

### 📜 Centralized Timeline

A dedicated history view records all interactions chronologically. Users can filter logs by interaction type and review detailed timestamps for each activity.

### 📊 Friendship Analytics

A dynamic Pie Chart visualizes communication patterns, updating instantly as new interactions are logged, giving insights into social behavior.

### 📱 Performance & UI/UX

Fully responsive design across all devices, custom loading states for better UX, and a dedicated 404 page for invalid routes.
